# 🏛️ Skill Meeting — AI 多角色圓桌討論平台

讓多個 AI 角色針對一個主題進行多輪圓桌討論，自動產出結構化報告。

---

## 這是什麼？

想像你有一個問題想討論，例如「如何提升用戶留存率？」。
這個工具會讓 4~6 個 AI 專家（產品經理、技術架構師、UX 研究員等）坐下來開會，
像真人一樣互相回應、辯論，最後自動產出一份有條理的報告。

### 內建的 AI 專家

| 角色 | 他會做什麼 |
|------|-----------|
| 🎯 產品經理 | 從用戶需求和商業價值的角度分析 |
| 🏗️ 技術架構師 | 從技術可行性和系統設計的角度分析 |
| 😈 魔鬼代言人 | 專門挑毛病、找漏洞，幫你避免盲點 |
| 🧑‍🎨 UX 研究員 | 從使用者體驗的角度分析 |

你也可以自己建立新的 AI 角色！

## 在你的電腦上安裝（一步一步來）

### 第一步：安裝 Node.js

Node.js 是讓這個程式能運行的工具，就像手機需要作業系統一樣。

1. 打開瀏覽器，前往 https://nodejs.org
2. 點擊綠色的 **「LTS」** 按鈕下載（選左邊那個）
3. 下載完成後，雙擊安裝檔，一路按「下一步」直到完成

**確認安裝成功：**
- Windows：按 `Win + R`，輸入 `cmd`，按 Enter 打開命令提示字元
- Mac：打開「終端機」（在 應用程式 > 工具程式 裡）

輸入以下指令，按 Enter：
```
node --version
```
如果看到類似 `v20.xx.x` 的數字，代表安裝成功！

---

### 第二步：取得 Anthropic API Key

API Key 就像一把鑰匙，讓程式能使用 Claude AI 的功能。

1. 前往 https://console.anthropic.com
2. 註冊或登入帳號
3. 點選左側的 **「API Keys」**
4. 點 **「Create Key」** 建立一把新鑰匙
5. **複製這串鑰匙**（長得像 `sk-ant-api03-...`），等一下會用到

⚠️ 注意：使用 API 會產生費用，但一場討論大約只需要幾塊台幣。

---

### 第三步：下載程式碼

1. 前往這個專案的 GitHub 頁面
2. 點綠色的 **「Code」** 按鈕，選 **「Download ZIP」**
3. 解壓縮到你喜歡的位置（例如桌面）

或者，如果你有裝 Git，也可以在命令列輸入：
```
git clone https://github.com/gjj22622/skill-meeting.git
```

---

### 第四步：安裝程式所需的套件

1. 打開命令列（Windows: cmd / Mac: 終端機）
2. 輸入以下指令，進入程式資料夾：

   **如果你解壓縮在桌面：**
   ```
   cd Desktop/skill-meeting
   ```

3. 輸入以下指令，安裝套件（需要等 1~2 分鐘）：
   ```
   npm install
   ```
   看到一堆文字跑完，沒有紅色的 `ERR` 就代表成功。

---

### 第五步：設定你的 API Key

1. 在程式資料夾裡，找到 `.env.example` 這個檔案
2. **複製一份**，把新檔案改名為 `.env.local`
3. 用記事本（或任何文字編輯器）打開 `.env.local`
4. 把裡面的 `sk-ant-your-api-key-here` 換成你在第二步複製的 API Key
5. 儲存檔案

完成後，`.env.local` 的內容應該長這樣：
```
ANTHROPIC_API_KEY=sk-ant-api03-你的真實key在這裡
```

---

### 第六步：啟動程式 🚀

在命令列輸入：
```
npm run dev
```

等幾秒鐘，看到這行字就代表成功了：
```
✓ Ready in XXXms
```

然後打開瀏覽器，在網址列輸入：
```
http://localhost:3000
```

🎉 **你應該會看到 Skill Meeting 的首頁了！**

## 怎麼使用？

### 1️⃣ 建立會議
- 點首頁的 **「+ 新會議」** 按鈕
- 輸入你想討論的主題（例如：「公司該不該導入 AI 工具？」）
- 可以貼上相關資料（可選）
- 選擇討論目標：達成共識 / 探索問題 / 腦力激盪
- 選擇討論輪數（建議 2~3 輪）
- 勾選你想參與討論的 AI 角色（至少 2 個，最多 6 個）
- 按 **「建立會議」**

### 2️⃣ 開始討論
- 進入會議室後，按 **「🚀 開始討論」**
- 你會看到 AI 角色們一個接一個發言
- 每個角色會有發言指示燈，表示誰正在說話
- 上方有進度條顯示討論進度

### 3️⃣ 查看報告
- 討論結束後，頁面下方會自動出現討論報告
- 報告包含：共識結論、分歧觀點、待釐清問題
- 每位 AI 角色都會在報告上留下自己的「簽名」
- 點 **「⬇️ 下載報告」** 可以匯出 Markdown 格式的完整報告

### 4️⃣ 管理 AI 角色
- 點導航列的 **「Skill 管理」**
- 可以查看所有角色
- 點 **「+ 建立 Skill」** 可以自己設計新角色
- 也可以用 **「📄 匯入 JSON」** 匯入別人分享的角色

---

## 要關閉程式？

回到命令列，按 `Ctrl + C` 就可以停止程式。
下次要再使用，只需要：
1. 打開命令列
2. 進入程式資料夾：`cd Desktop/skill-meeting`
3. 輸入：`npm run dev`
4. 打開瀏覽器輸入：`http://localhost:3000`

## 常見問題

**Q：打開 localhost:3000 是空白頁面？**
A：等幾秒鐘再重新整理，第一次載入需要一點時間。

**Q：按「開始討論」沒反應或出現錯誤？**
A：請確認 `.env.local` 裡的 API Key 是正確的，而且帳號裡有餘額。

**Q：怎麼更新程式？**
A：重新從 GitHub 下載最新版本，再執行一次 `npm install` 就好。
