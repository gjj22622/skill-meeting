# Skill Meeting PRD
版本：客戶版精簡稿
日期：2026-03-28

## 1. 產品概述
Skill Meeting 是一個 AI 多角色圓桌討論平台，讓使用者能為同一議題指派多位 AI 角色進行多輪討論，並自動生成結構化報告。

它的核心價值不是單次問答，而是把一個問題拆解成多個專業視角，模擬真實會議中的觀點交流、質疑、補充與收斂，適合用於企劃討論、商業分析、策略發想、教學設計與顧問提案準備。

## 2. 產品目標
1. 讓使用者快速獲得多角色、多觀點的 AI 討論結果。
2. 讓企業或個人能建立自己的 AI 角色庫，沉澱專業方法論。
3. 讓討論結果自動整理為可保存、可下載、可複製的報告。
4. 進一步發展為可分享與安裝 AI 角色的 Skill Marketplace。

## 3. 目標使用者
1. 顧問、企劃與策略人員。
2. 企業內部專案團隊與主管。
3. 教學培訓與工作坊設計者。
4. 想建立專業 AI 角色庫的進階使用者。

## 4. 核心使用流程
1. 建立新會議並輸入討論主題。
2. 補充背景資料或參考資訊。
3. 選擇參與討論的 AI Skill。
4. 設定討論目標與輪數。
5. 啟動多角色討論。
6. 查看即時對話與最終報告。
7. 下載 Markdown 報告或持續複用既有角色。

## 5. 核心功能需求
### 5.1 AI 多角色討論
系統支援 2 至 6 位 AI 角色參與同一場討論，針對單一主題依序輸出觀點，並進行多輪回應。

支援的討論目標包含：
1. 達成共識
2. 探索問題
3. 腦力激盪

### 5.2 Skill 管理
Skill 為可重複使用的 AI 角色設定，包含角色名稱、頭像、專長、人格描述、System Prompt 與簽名樣式。

目前產品支援：
1. 預設 Skill 庫
2. 自訂 Skill 建立
3. JSON 匯入
4. 啟用與停用管理
5. 自訂 Skill 刪除

### 5.3 即時會議室
會議室以多角色發言流程呈現討論過程，包含：
1. 目前發言角色
2. 討論階段
3. 討論輪次
4. 進度條
5. 即時訊息串流

### 5.4 結構化報告
每場討論完成後，自動產出結構化報告，內容包含：
1. 會議資訊
2. 共識結論
3. 分歧觀點
4. 待釐清問題
5. 完整對話記錄
6. 各角色簽署

並支援匯出為 Markdown。

### 5.5 Skill Marketplace
Marketplace 是產品的延伸模組，讓使用者可以發現、安裝與發佈 Skill。

目前規劃方向包含：
1. 市集首頁與精選內容
2. 搜尋與分類
3. Skill 詳情頁
4. 安裝到本地角色庫
5. 發佈新的 Marketplace Skill

## 6. 目前版本交付範圍
目前版本已可提供以下體驗：
1. 建立新會議
2. 選擇多位 Skill 進行多輪討論
3. 查看即時討論內容
4. 生成並下載討論報告
5. 管理預設與自訂 Skill
6. 瀏覽與安裝 Marketplace Skill
7. 發佈新的 Marketplace Skill
8. 使用 SOSTAC 類型的擴充角色進行策略討論

## 7. 現階段定位
目前產品定位為可操作、可展示的 MVP 版本，已完成主要使用流程與核心差異化體驗，適合用於：
1. 客戶展示
2. 概念驗證
3. 內部試用
4. 下一階段正式產品化評估

## 8. 下一階段建議
### Phase 1：產品正式化
1. 建立使用者登入與權限系統
2. 導入正式資料庫
3. 完成會議、Skill、Marketplace 的後端資料管理
4. 補強穩定性、驗證與錯誤處理

### Phase 2：Marketplace 平台化
1. 作者頁與帳號身份
2. Skill 版本管理
3. 評價與下載統計
4. 上架審核機制

### Phase 3：商業化
1. 付費 Skill
2. 團隊共享角色庫
3. 企業版工作區
4. 白牌化與客戶專屬模板

## 9. 專案價值摘要
Skill Meeting 不只是「和 AI 對話」，而是把 AI 角色化、流程化、資產化。

它同時具備三層價值：
1. 討論工具：幫助使用者快速獲得多方觀點。
2. 方法論工具：把企業自己的角色與分析框架沉澱成 Skill。
3. 平台工具：未來可擴展為 Skill 發佈、流通與商業化平台。
